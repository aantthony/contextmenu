ContextMenu

Responsive and native-looking javascript right click menu

Introduction

Contextmenu is a javascript library (with no dependencies) that allows the creation of native-looking right-click menus within a web page. Don't use this in an attempt to replace the default context menu (e.g., to prevent Download Linked File As..., Save Image As..) because that is very annoying. The intended use is for creating menus on objects within a page (not the entire page), and for javascript-heavy web apps.

Instructions

Download the contextmenu.css and contextmenu.js files (above)

Then in your page, add the following:

<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="contextmenu.css" type="text/css" media="all" />
    ...
  </head>
  <body>
    ...
    <script type="text/javascript" src="contextmenu.js"></script>
  </body>
</html>

Either specify the markup using HTML context-menu notation, or use the simple Javascript API provided by the contextmenu object which builds DOM nodes, and adds some javascript to ensure browser compatibility.

API Example

var menu = contextmenu([
  {
    label: "First Item",
    onclick: function (e){
      document.body.style.background = "black";
      setTimeout(function() {
        document.body.style.background = "";
      }, 500);
    }
  },
  {label: "Second Item"},
  {hr : true},
  {
    label: "Sub menu",
    children: [
      {
        label: "Another Item"
      }
    ]
  }
]);

contextmenu.attach(domNode, menu);

contextmenu.show(menu, 300, 200);

Detailed Example

Javascript API equivalent 
var menu = contextmenu([
  {label: "Standard"},
  {hr: true},
  {label: "Checkbox", checked: true},
  {label: "Key", key: "\u2318A"}
]);
contextmenu.attach($("#id"), menu);

This will work independently of whether executed before or after the page loads.

HTML notation: (direct child of body) 
<menu type="context" id="menu">
  <menuitem label="Standard"></menuitem>
  <hr />
  <menuitem label="Checkbox" checked="checked"></menuitem>
  <menuitem label="Key" key="&#8984;A"></menuitem>
</menu>

If the html node is generated after the page load, then call contextmenu.prepare($("#menu")); and use contextmenu.attach when needed. 

        
<div contextmenu="menu" id="id" />
